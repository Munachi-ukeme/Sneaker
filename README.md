Sneakers is a one‑page company website built entirely with HTML and CSS. It showcases a modern sneaker brand with clean design, product collections, and essential company information.

Sneakers was created as a beginner front‑end project to practice building a brand landing page.
It demonstrates how a single page can present product collections, company details, and contact information using only HTML and CSS.

Features:
1. Single Page Layout: All content presented on one HTML page

2. Collection Section: Highlighting featured sneakers

3. Men’s Section: Showcasing sneakers designed for men

4. Women’s Section: Showcasing sneakers designed for women

5. About Section: Company story and brand identity

6. Contact Section: Static contact form or details for reaching out

Lessons Learned:
This project helped me practice:
1. Designing a one‑page layout with semantic HTML

2. Styling product sections with CSS

3. Creating a navigation bar for smooth scrolling

4. Documenting projects professionally on GitHub

Navigation Bar: Smooth navigation across sections (Collection, Men, Women, About, Contact)

CSS Styling: Clean, modern design with responsive elements
